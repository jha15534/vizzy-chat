# Vizzy AI

Vizzy AI is an AI-powered web application that enables users to chat with AI, generate AI images, and manage conversations through a modern and responsive interface. The project is built using React, FastAPI, MySQL, and Google Gemini AI.

---

## Features

- User Authentication (Signup & Login)
- AI Chat Assistant
- AI Image Generation
- Markdown Response Support
- Chat History
- Delete Chat
- Responsive User Interface
- Modern Dashboard
- Google Gemini AI Integration

---

## Installation

### Clone Repository

```bash
git clone <repository-url>
cd vizzy-chat
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
